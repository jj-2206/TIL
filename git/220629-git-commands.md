# 오늘 한 일
- shell, vim 커맨드
  - cd 경로 이동
  - mkdir 디렉토리 (폴더) 생성
  - touch 파일 생성
  - mv 파일 이동 ex. mv readme.md bin/ bin 폴더로 readme.md 파일 이동
  - cp 파일 복사 ex. cp readme.md bin/ bin 폴더에서 readme.md 복사
  - mv 이름 변경 ex. mv readme.md ./README.txt
  - rm 파일 삭제 ex. rm README.txt
  - rm -rf bin/ 강제 삭제
  - vi 파일 작성
  - cat 파일 내용 확인

- git 커맨드
  - git clone {repo url}
  - git add {filename}
  - git commit
  - git push origin main
