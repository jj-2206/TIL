# 오늘의 코드

내장 객체들에 익숙해져야한다...

```js
const original = [{ a: 1 }, { a: 5 }, { a: 2 }, { a: 3 }];
const mapFunction = (item, index) => {
  let value = item.a;
  return { a: value * 2 };
};
const a = original.map(mapFunction);
console.log(a);
```
